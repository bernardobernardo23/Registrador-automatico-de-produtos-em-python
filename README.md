 Esse projeto utiliza uma versão com alterações do projeto disponibilizado no video https://www.youtube.com/watch?v=pNBjC32nisg

   O registrador automático utiliza de atalhos de teclado diferentemente do video onde é usado o mouse,
 por conta de diferentes tipos de abertura de arquivo e tamanhos de tela diferente o uso do mouse pode nao ser a melhor maneira 
 para um programa como esse.
   É recomendado para o funcionamento que a pasta do projeto seja fixada no acesso rápido do explorador de arquivos, para que sempre
que o registrador abrir com win+e a pasta estar ali na página inicial
